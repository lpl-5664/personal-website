Language Translation Web Application

Transformer Neural Machine Translation

A deep learning-based translation engine developed using Python and TensorFlow, leveraging a Transformer model similar to Google's neural machine translation system. Designed for high efficiency, it powers the Language Translation Web Application by serving as its core translation component.

Multi-Object Tracker

A computer vision project implementing a Multi-Object Tracking (MOT) system using deep learning. Built with YOLOv4 for real-time object detection and a Kalman filter for predictive tracking, the system identifies and tracks multiple objects in video frames with high accuracy. The project utilizes TensorFlow and OpenCV for video preprocessing, making it suitable for surveillance, autonomous vehicles, and smart traffic systems.

Technologies: Python, MATLAB, Tensorflow, OpenCV
